# Group 1 | Talenthunt

## Deployment for frontend

To run frontend react server

cd frontend

First, install the node module following this command:

```bash
  npm install
```

Then run

```bash
  npm start
```

## Deployment for backend

To run backend node server

cd backend

First, install the node module following this command:

```bash
  npm install
```

Then run

```bash
  npm start
```
